Applying Computational Science and Engineering

Module synopsis

The aim of this module is to simulate applied computational science in the real world. The module comprises four group projects that synthesise different elements of the MSc program. Each project will span five days of full-time effort, simulating a working week. On Monday morning, groups will meet with the project “client”. The client will outline the problem to be solved, and be available for one-to-one meetings with the groups to specify the project scope. On Friday afternoon, the groups will deliver or present their work to the client for assessment. Each project will be assessed based on the group presentation; the software produced (e.g., functionality, robustness, ease-of-use, documentation); and the software’s performance in the specified tasks.

Module Objective

  • To simulate applied computational science in the real world.
  • To synthesise knowledge from the taught modules.
  • To apply techniques learned in the course to real problems.
  • To develop collaborative programming skills.
  • To reinforce best practise for software development.

Learning Outcome

  • Plan and produce software collaboratively.
  • Collaboratively write technical reports.
  • Summarise technical reports through collaborative presentations.
